# shiny90
Shiny tool for the first 90 HIV model (https://github.com/mrc-ide/first90)

## Prerequisites
* R

## Install dependencies
Run

```
./bootstrap
```

to install R dependencies. If your system doesn't suppport bash, then instead run `./bootstrap.R`.

## Run app
Then run 

```
./app.R
```

to run the app.
